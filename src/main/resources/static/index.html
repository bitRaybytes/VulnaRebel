Objective:
VulnaRebel is a deliberately vulnerable web application designed for learning and practicing web application security. Each challenge simulates a realistic vulnerability and has one objective: capture the hidden flag (ctf{...}), which may be located on either the client side or the server side.

If you get stuck, a built-in hint system is available to guide you. Before revealing the next hint, try to investigate the behavior yourself and research the underlying vulnerability using your preferred search engine. Understanding why an attack works is more valuable than simply finding the flag.
